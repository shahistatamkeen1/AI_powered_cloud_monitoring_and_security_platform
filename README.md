# 🚀 AI-Powered Cloud Monitoring and Security Platform

## 📌 Overview

This project is a cloud monitoring system that provides a centralized dashboard to track system performance, alerts, and logs. It uses AI to simplify complex issues and suggest solutions.

---

## 🎯 Features

* 📊 **Dashboard** – View CPU, Memory, and Network usage with charts
* 🚨 **Alerts** – Real-time alerts with acknowledge & resolve actions
* 📜 **Logs** – Centralized system logs
* 🤖 **AI Insights** – Simple explanations + suggested fixes

---

## 🧑‍💻 Tech Stack

* **Frontend:** React (Vite), Tailwind CSS
* **Backend:** FastAPI / Node.js (planned)
* **Database:** PostgreSQL (pgAdmin)
* **AI:** OpenAI (planned)

## ⚙️ Setup
